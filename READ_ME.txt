READ ME!

This project is based on metro ticket purchasing.

Facilities provided:
1) User Registration
2) User Login and Logout
3) Ticket Purchase
4) View Cart
	4.1) Delete from Cart
5) Print Ticket
6) Administration
	6.1) Ticket Administration
	6.2) Zone Administration

Project implemented using Kohana Framework. The project utilizes Model View Controller framework for page navigation and its business logic.
For the data layer: RedBeanPhp has been implmented.
For the presentation layer: Bootstrap, twig and Jquery has been implemented.
